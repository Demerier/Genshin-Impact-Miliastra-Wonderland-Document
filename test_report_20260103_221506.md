# 一、背包的定义

背包是整个资源系统的容器，背包管理持有者的所有虚拟道具、装备和货币。

在背包界面中，道具以槽位的方式进行展示，玩家可以在运行时对背包内的道具进行拆分，销毁或丢弃操作

打开系统菜单，通过点击【货币与背包】按钮进入编辑界面

**基础属性**

* *背包格数量*：初始的背包内栏位数量，每一个背包栏位可以容纳一种道具，并根据堆叠数量可能堆叠多个
