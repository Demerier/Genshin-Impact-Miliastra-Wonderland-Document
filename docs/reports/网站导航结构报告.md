## 4. 爬取建议

### 4.1 爬取策略
1. **按分类爬取**：先爬取主分类，再爬取各主分类下的子分类
2. **并行爬取**：使用异步编程模型，提高爬取效率
3. **深度优先**：优先爬取完整的知识领域，再爬取下一个领域
4. **断点续爬**：记录已爬取的文档ID，支持断点续爬

### 4.2 链接处理
1. **链接提取**：从导航结构中提取所有文档URL
2. **去重处理**：确保每个文档只爬取一次
3. **本地化转换**：将绝对URL转换为本地文件引用
4. **映射表维护**：建立文档ID与中文标题的映射关系

### 4.3 内容提取
1. **定位内容区**：使用`.doc-view`类定位文档内容
2. **保留结构**：完整保留标题层级、段落和链接
3. **转换格式**：将HTML转换为Markdown格式
4. **处理图片**：下载并本地化所有图片资源
