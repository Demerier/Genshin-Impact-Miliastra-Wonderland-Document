## 路径


# 一、路径的功能

路径是将一至多个只包含位置和旋转信息的<span style="color: blue; font-style: italic;">路点</span>，汇总为一组路径信息的功能。

路径中的路点信息为世界坐标和旋转，不会因为被引用而被调整数据内容。

路径可被任意支持功能所引用，目前包括<span style="color: red; font-style: italic;">造物</span>的<span style="color: red; font-style: italic;">未入战-巡逻</span>功能、基础运动器组件的关卡路径运动。

# 二、路径管理工具的功能

<span style="color: blue; font-style: italic;">路径</span>通过路径管理工具进行统一管理

可以新增/删除路径，以及对路径内数据进行调整

可以管理路径的页签

# **三、路径的编辑**

## 1.入口

打开左上角系统菜单，选择<span style="color: blue; font-style: italic;">路径管理</span>，打开路径管理工具

![](../images/路径_mh333vim2h44_1入口.png)

## 2.工具展示界面

分为路径管理栏、路径配置栏

![](../images/路径_mh333vim2h44_2工具展示界面.png)

### **(1)路径管理栏**

![](../images/路径_mh333vim2h44_1路径管理栏.png)

* 新建路径

通过【创建路径】可以新建路径

新建的路径默认属于<span style="color: blue; font-style: italic;">未分类页签</span>

并以屏幕中心点作为该路径的第一个路点

![](../images/路径_mh333vim2h44_1路径管理栏_3.png)

![](../images/路径_mh333vim2h44_1路径管理栏_4.png)

* 管理页签

选择<span style="color: blue; font-style: italic;">未分类页签</span>的可以新增页签，并对新增的页签命名

![](../images/路径_mh333vim2h44_1路径管理栏_6.png)

![](../images/路径_mh333vim2h44_1路径管理栏_7.png)

新增页签默认内容为空，可以指定路径，右键选择<span style="color: blue; font-style: italic;">更改页签</span>，调整路径所在的页签，便于管理

![](../images/路径_mh333vim2h44_1路径管理栏_8.png)

![](../images/路径_mh333vim2h44_1路径管理栏_9.png)![](../images/路径_mh333vim2h44_1路径管理栏_10.png)

### **(2)路径配置栏**

![](../images/路径_mh333vim2h44_2路径配置栏.png)

* 索引和命名

![](../images/路径_mh333vim2h44_2路径配置栏_2.png)


|  |  |
| --- | --- |
| 配置参数 | 说明 |
| <span style="color: blue; font-style: italic;">\名称</span> | 可用于标识的路径名称，不可重复，可修改 |
| <span style="color: blue; font-style: italic;">索引</span> | 路径的唯一标识，可用作节点图入参 |


* 基础配置

![](../images/路径_mh333vim2h44_2路径配置栏_3.png)


|  |  |
| --- | --- |
| 配置参数 | 说明 |
| <span style="color: blue; font-style: italic;">\锁定变换</span> | 若开启，则该路径只能新增和删除路点，不可调节任意路点的信息 |
| <span style="color: blue; font-style: italic;">\是否在场景中显示</span> | 若开启，则场景中会显示该路径的所有路点和连接顺序 |
| <span style="color: blue; font-style: italic;">\路径中心点位置</span> | 路径的中心点位置，这个位置和所有路点的相对位置会保持不变。因此修改该位置可以影响该路径的所有路点的位置 |
| <span style="color: blue; font-style: italic;">\路径中心点旋转</span> | 路径的中心点旋转，这个旋转和所有路点的相对旋转会保持不变。因此修改该旋转可以影响该路径的所有路点的旋转 |


* 路点列表

![](../images/路径_mh333vim2h44_2路径配置栏_5.png)


|  |  |
| --- | --- |
| 配置参数 | 说明 |
| <span style="color: blue; font-style: italic;">位置</span> | 可手动调整参数，修正路点位置 |
| <span style="color: blue; font-style: italic;">旋转</span> | 可手动调整参数，修正路点旋转 |
| <span style="color: blue; font-style: italic;">\添加路点</span> | 点击创建新路点 |
| <span style="color: blue; font-style: italic;">\自由放置</span> | 点击后编辑界面清空，通过鼠标左键点击场景，创建新路点  通过可以结束新建路点操作 |

*点击路点的右侧按钮可展开辅助编辑

![](../images/路径_mh333vim2h44_2路径配置栏_10.png)

![](../images/路径_mh333vim2h44_2路径配置栏_11.png)


|  |  |
| --- | --- |
| 配置参数 | 说明 |
| <span style="color: blue; font-style: italic;">指向终点方向切线</span> | 调整路点旋转指向下一个路点 |
| <span style="color: blue; font-style: italic;">指向起点方向切线</span> | 调整路点旋转指向上一个路点 |
| <span style="color: blue; font-style: italic;">全选路径内路点</span> | 选中该路径上的所有路径点，若没有路径点则置灰 |
| <span style="color: blue; font-style: italic;">向终点方向新增路点</span> | 路点列表，在该点的序号后添加一个路点。  例如在2号路点添加一个后序路点，新增路点为3号路点，之前的3号路点会成为4号路点 |
| <span style="color: blue; font-style: italic;">向起点方向新增路点</span> | 路点列表，在该点的序号前添加一个路点。  例如在2号路点添加一个前序路点，新增路点为2号路点，之前的2号路点会成为3号路点 |
| <span style="color: blue; font-style: italic;">复制</span> | 复制路点的坐标和旋转信息 |
| <span style="color: blue; font-style: italic;">粘贴</span> | 粘贴已复制的路点坐标和旋转信息 |
| <span style="color: blue; font-style: italic;">删除该路点</span> | 删除指定路点 |


* 引用溯源

![](../images/路径_mh333vim2h44_2路径配置栏_12.png)


|  |  |
| --- | --- |
| 配置参数 | 说明 |
| <span style="color: blue; font-style: italic;">\造物引用</span> | 显示该路径被引用次数  可定位引用该路径的造物 |
| <span style="color: blue; font-style: italic;">\物件引用</span> | 显示该路径被引用次数  可定位引用该路径的物件 |
