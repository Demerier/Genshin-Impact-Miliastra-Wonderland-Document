## 计分板界面控件


# 一、计分板的功能

![](../images/计分板界面控件_mhesro0hyn5k_一计分板的功能.png)

关卡运行时，用于显示玩家自定义变量，并可以根据自定义变量进行排名的信息展示界面。

# **二、计分板的编辑**

![](../images/计分板界面控件_mhesro0hyn5k_二计分板的编辑.png)


|  |  |
| --- | --- |
| 配置参数 | 说明 |
| <span style="color: blue; font-style: italic;">排序</span> | 提供数值升序和降序 |
| <span style="color: blue; font-style: italic;">类型</span> | 个人、阵营，类型的选择将影响余下配置参数的配置内容 |


# **三、计分板的类型**


|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 类型 | 计分板样式 | 样式设置项 | 数据设置项 | 说明 |
| <span style="color: blue; font-style: italic;">个人</span> |  |  |  | 可设置计分板中需要显示的内容，包含名次/分数/阵营图标/玩家名称/玩家头像 |
| <span style="color: blue; font-style: italic;">阵营</span> |  |  |  | 可设置积分板中需要显示的内容，包含名次/分数 |
