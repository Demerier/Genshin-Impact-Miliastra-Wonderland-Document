# 一、战斗设置是什么

提供自定义战斗玩法设计更详细的配置，是玩家设置攻击与受击相关信息的手段

# 二、战斗设置编辑

在特化配置页签下，可点击*详细编辑*进入实体的战斗设置

![](https://act-webstatic.mihoyo.com/ugc-tutorial/knowledge/cn/zh-cn/mha42r0cwx74/e73e4178-bfef-4f8f-834e-bb450d10dfa3.png)

|  |  |
| --- | --- |
| 配置参数 | 说明 |
| *不可被元素附着* | 配置该物件是否会被元素附着 |
| *不可被目标锁定* | 配置任何锁定该物件的功能是否会生效 |
| *受击特效* | 配置物件受到攻击时播放的特效 |
| *被击倒特效* | 配置物件被销毁或倒下时播放的特效 |

其中受击特效和被击倒特效可以在详细编辑页面进一步配置

![]()

|  |  |
| --- | --- |
| 特效配置参数 | 说明 |
| *特效资产类型* | 仅可选*限时特效*  特效相关资产见[特效](/ys/ugc/tutorial//detail/mhe1030vx380) |
| *特效资产* | 可以选择特效资产。选择了特效资产后，可以在场景中预览播放 |
| *缩放比例* | 特效的缩放比例 |
| *偏移* | 指定相对位置 |
| *旋转* | 指定特效的旋转信息 |
