## 战斗设置


# 一、战斗设置是什么

提供自定义战斗玩法设计更详细的配置，是玩家设置攻击与受击相关信息的手段

# 二、战斗设置编辑

在特化配置页签下，可点击<span style="color: blue; font-style: italic;">详细编辑</span>进入实体的战斗设置

![](../images/战斗设置_mha42r0cwx74_二战斗设置编辑.png)


|  |  |
| --- | --- |
| 配置参数 | 说明 |
| <span style="color: blue; font-style: italic;">不可被元素附着</span> | 配置该物件是否会被元素附着 |
| <span style="color: blue; font-style: italic;">不可被目标锁定</span> | 配置任何锁定该物件的功能是否会生效 |
| <span style="color: blue; font-style: italic;">受击特效</span><span style="color: blue;"> </span> | 配置物件受到攻击时播放的特效 |
| <span style="color: blue; font-style: italic;">被击倒特效</span>* * | 配置物件被销毁或倒下时播放的特效 |

*其中受击特效和被击倒特效可以在详细编辑页面进一步配置

![](../images/战斗设置_mha42r0cwx74_二战斗设置编辑_2.png)


|  |  |
| --- | --- |
| 特效配置参数 | 说明 |
| <span style="color: blue; font-style: italic;">特效资产类型</span> | 仅可选*限时特效*  特效相关资产见[特效](特效_mhe1030vx380.md) |
| <span style="color: blue; font-style: italic;">特效资产</span> | 可以选择特效资产。选择了特效资产后，可以在场景中预览播放 |
| <span style="color: blue; font-style: italic;">缩放比例</span><span style="color: blue;"> </span> | 特效的缩放比例 |
| <span style="color: blue; font-style: italic;">偏移</span><span style="color: blue;"> </span> | 指定相对位置 |
| <span style="color: blue; font-style: italic;">旋转</span> | 指定特效的旋转信息 |
