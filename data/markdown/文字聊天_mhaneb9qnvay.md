## 文字聊天


# 一、文字聊天功能

文字聊天管理提供贴合玩法的聊天功能，支持根据玩法需要配置多个针对不同玩家或阵营的专属聊天频道。

同一个玩家可以同时在不同的聊天频道中发言

支持利用节点图实时切换玩家聊天频道的权限。

# 二、文字聊天功能管理

![](../images/文字聊天_mhaneb9qnvay_二文字聊天功能管理.png)

通过【系统菜单】——【文字聊天管理】可管理当前关卡聊天频道

![](../images/文字聊天_mhaneb9qnvay_二文字聊天功能管理_2.png)

* 全局设置：

 控制当前关卡内文字聊天功能是否开启

# 三、文字聊天频道的编辑

* 添加聊天频道

通过【添加频道】按键，可新增聊天频道

关卡默认带有应用于全部玩家的全局频道，支持修改配置/改名，但无法删除

![](../images/文字聊天_mhaneb9qnvay_三文字聊天频道的编辑.png)

点击 图标，可重命名，或删除频道

* 基础信息

![](../images/文字聊天_mhaneb9qnvay_三文字聊天频道的编辑_3.png)


|  |  |
| --- | --- |
| 参数 | 说明 |
| <span style="color: blue; font-style: italic;">索引</span> | 对应聊天频道的唯一标识 |
| <span style="color: blue; font-style: italic;">初始生效</span> | 控制频道在关卡初始时的生效情况 |
| <span style="color: blue; font-style: italic;">频道应用方式</span> | 支持按玩家/按阵营应用  当频道按阵营应用时，仅可通过切换玩家阵营调整所属聊天频道 |
| <span style="color: blue; font-style: italic;">应用玩家/阵营</span> | 控制频道在关卡内的初始覆盖对象，可多选 |
| <span style="color: blue; font-style: italic;">显示优先级</span> | 控制频道在关卡内聊天系统的显示顺序，数字越大越靠前 |
| <span style="color: blue; font-style: italic;">选择图标</span> | 控制频道在关卡内聊天系统的图标与颜色 |


* 快捷消息

控制对应频道在关卡内的快捷聊天内容

点击     可添加快捷文本，点击   可删除文本，至少需要保留1条默认文本

# 四、以节点图控制文字聊天频道

* 设置玩家当前频道

通过节点图输入按玩家划分的频道索引列表，可覆盖目标玩家的按玩家划分的生效频道

![](../images/文字聊天_mhaneb9qnvay_四以节点图控制文字聊天频道.png)

* 设置聊天频道开关

通过节点图可控制当前关卡内所有文字聊天频道开关，开启/关闭可对该频道所有对应对象生效

* 修改玩家频道权限

通过节点图输入可将特定玩家从某个按玩家划分的频道中踢出或拉入

![](../images/文字聊天_mhaneb9qnvay_四以节点图控制文字聊天频道_3.png)
