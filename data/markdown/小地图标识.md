# 一、小地图标识组件的功能

*小地图标识组件*，支持实体通过图标/范围等可视化标识显示在**预制小地图界面控件**中

小地图标识组件需要挂载于实体，支持*玩家*、*物件*、*造物*使用

小地图标识可以通过默认配置激活/关闭，也可以通过*节点图*控制指定小地图标识的激活/关闭

# **二、**小地图标识组件**的编**辑

## 4. 小地图标识的参数说明

### (1)显示设置

![]()

|  |  |  |
| --- | --- | --- |
| 配置参数 | 说明 | |
| *初始所有玩家可见* | 若开启，则小地图标识生效时，满足条件的所有玩家都可以在其小地图界面控件上看到该标识 | |
| *跟随物体可见性* | 若开启，则实体隐藏时，小地图标识一同隐藏 | |
| *显示优先级* | 在同一位置的小地图标识，造成堆叠后，优先级更高的会覆盖优先级低的。数字越大，优先级越高 | |

### (2)标记样式

![]()

|  |  |
| --- | --- |
| 配置参数 | 说明 |
| *选择类型* | 分为图标、范围、标记点、玩家标记、造物头像几种，不同类型会带有不同的配置参数 |
| *显示高低差* | 开启后，高低差超过10m的实体会在小地图内额外标记 |

### (3)选择类型的分类



图标

![]()

|  |  |
| --- | --- |
| 配置参数 | 说明 |
| *选择图标* | 提供多种可供选择的图标 |
| *底图* | 提供无、圆形 |
| *底图外框颜色* | 图标底色支持选择，包括指定颜色和逻辑颜色。  逻辑颜色包括  【敌友关系】敌方红色、友方绿色、自身蓝色  【跟随自身阵营】根据自身当前的阵营颜色  【跟随所有者阵营】跟随当前实体的所有者阵营颜色 |
| *是否可点击* | 开启后，展开地图，点击标识点，会显示配置信息 |
| *文本内容* | 支持配置点击后的右侧显示文本 |



范围

![]()

|  |  |
| --- | --- |
| 配置参数 | 说明 |

| *颜色* | 图标底色支持选择，包括指定颜色和逻辑颜色。  逻辑颜色包括  【敌友关系】敌方红色、友方绿色、自身蓝色  【跟随自身阵营】根据自身当前的阵营颜色  【跟随所有者阵营】跟随当前实体的所有者阵营颜色 |
| *范围大小(m）* | 根据配置的尺寸，在小地图界面控件中，按比例显示 |
