## 界面布局


# 一、界面布局的定义

界面布局是关卡运行时，玩家用于进行信息获取和交互操作的界面。

在包含基础功能的*默认布局*之上，支持玩家对界面布局自定义。

界面布局依赖玩家运行，运行时每个玩家有且仅有一个生效的界面布局。

# **二、界面布局的管理**

![](../images/界面布局_mhozt0r74ng6_二界面布局的管理.png)

界面布局通过*界面控件组管理-界面布局*统一管理。

仅支持对关卡运行中表现的界面做编辑和引用，并不支持对匹配、结算等必要流程界面的编辑。

## 1.默认配置

玩家在关卡游玩时使用的界面布局，需要通过*职业*进行配置。

* *打开“职业”编辑界面，通过<span style="color: blue; font-style: italic;">选择布局</span>，引用界面布局*

![](../images/界面布局_mhozt0r74ng6_1默认配置.png)

* 界面布局的参数配置窗口也可以直接跳转到该编辑界面

![](../images/界面布局_mhozt0r74ng6_1默认配置_2.png)

## 2.以节点图管理

* 切换当前界面布局

通过布局索引，将目标玩家当前的界面布局按需替换。

![](../images/界面布局_mhozt0r74ng6_2以节点图管理.png)


|  |  |
| --- | --- |
| 配置参数 | 说明 |
| <span style="color: blue; font-style: italic;">目标玩家</span> | 需要切换界面布局的玩家实体 |
| <span style="color: blue; font-style: italic;">布局索引</span> | 界面布局管理工具中，指定布局的索引 |


* 获取当前界面布局

运行时，获取玩家当前的界面布局索引。

# **三、界面布局的适配说明**

## 1.选择设备

由于不同设备上的分辨率/按钮排布等界面布局差异较大，如果需要制作<span style="color: blue; font-style: italic;">PC端/移动端/主机端</span>都可以游玩的关卡，则需要调整控件在不同设备上的<span style="color: blue; font-style: italic;">位置、大小、锚点</span>

可调整设备的位置有两处：

(1)顶部栏：可选择设备及分辨率

![](../images/界面布局_mhozt0r74ng6_1选择设备.png)

(2)控件详情-变换：点击选择设备

## 2.调整位置与大小

### (1)位置

以画布左下角为原点，显示控件中心点与原点的相对位置

每个设备需单独设置

![](../images/界面布局_mhozt0r74ng6_1位置.png)

### (2)大小

调整控件当前大小，点击右上角链条按钮可锁定当前长宽比，部分控件大小不可调整

每个设备需单独设置

![](../images/界面布局_mhozt0r74ng6_2大小.png)

## 3.锚点

### (1)定义

当分辨率变化时，控件会自适应位置与设置的锚点保持相对位置不变

以下图选中的任务文本为例：

![](../images/界面布局_mhozt0r74ng6_1定义.png)

| --- | --- |

### (2)**锚点选择建议**

* 根据控件所在位置：在九宫格中主体位于哪部分就选择对应位置的锚点

![](../images/界面布局_mhozt0r74ng6_2锚点选择建议.png)

* 根据功能区：比如任务描述，多条文本框的锚点选择应该保持一致

![](../images/界面布局_mhozt0r74ng6_2锚点选择建议_2.png)

## 4.界面死区

移动端界面最小，需要以移动端16:9适配作为基准确保可点击控件在红框区域内

![](../images/界面布局_mhozt0r74ng6_4界面死区.png)

# 四、界面规范参考

## 1.文字参考

### (1)字号

输入文本时建议参考下图选择字号，通常采用20号字

|  |  |
| --- | --- |
| 类型 | 推荐字号 |
| 大标题 | 36-32 |
| 小标题/按钮文本 | 24 |
| 正文<span style="color: blue;">（最常用）</span> | 20 |
| 辅助字 | 18 |
| 推荐最小字号 | 16 |
